# Проект "На ладони"

## Синхронизация с GitHub

Для синхронизации ваших изменений с GitHub:

### Быстрый способ
Используйте скрипт синхронизации:
```bash
./sync.sh
```

### Документация
- [SYNC_GUIDE.md](SYNC_GUIDE.md) - Полное руководство по синхронизации с GitHub
- [GIT_QUICK_REF.md](GIT_QUICK_REF.md) - Быстрая справка по командам Git

### Основные команды

Получить изменения с GitHub:
```bash
git pull origin main
```

Отправить изменения на GitHub:
```bash
git add .
git commit -m "Описание изменений"
git push origin main
```
