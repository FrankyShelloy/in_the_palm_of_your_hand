# Быстрая справка по Git

## Самые частые команды

### Получить изменения
```bash
git pull
```

### Отправить изменения
```bash
git add .
git commit -m "Сообщение"
git push
```

### Посмотреть статус
```bash
git status
```

### История коммитов
```bash
git log --oneline -10
```

### Создать ветку
```bash
git checkout -b новая-ветка
```

### Переключить ветку
```bash
git checkout имя-ветки
```

## Аварийные ситуации

### Отменить изменения в файле
```bash
git checkout -- имя-файла.cs
```

### Вернуться к предыдущему коммиту
```bash
git reset --hard HEAD~1
```

### Временно сохранить изменения
```bash
git stash
git stash pop  # восстановить
```

### При конфликте слияния
1. Откройте файл и найдите `<<<<<<<`
2. Исправьте код, удалите маркеры
3. `git add файл`
4. `git commit`

## Использование скриптов
